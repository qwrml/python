Инструкция для проверки домашнего задания

Чтобы запустить тесты:

1. Замените API ключ в файле config.py:
   - Откройте файл config.py
   - Найдите строку: API_KEY = "your_api_key_here"
   - Замените на ваш настоящий API ключ: API_KEY = "ваш_ключ_здесь"

2. Установите библиотеки:
   - Выполните в терминале: pip install -r requirements.txt

3. Запустите тесты:
   - Выполните в терминале: pytest

Где взять API ключ:
- Залогиньтесь в Yougile
- Получите ID компании
- Выполните API запрос для получения ключа (как в документации Yougile)
